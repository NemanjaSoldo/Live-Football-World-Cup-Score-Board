# Live Football World Cup Score Board - assignment

This application runs in a console where user can select options

Prerequisites:
1. Installed Node (v22.7)
2. npm (10.8)

Process:
1. Run (in the project root directory):
    ```sh
    npm install
    ```
2. To start the application run:
    ```sh
    npm start
    ```
3. To run tests:
    ```sh
    npm test
    ```
